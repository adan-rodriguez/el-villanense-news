## Run Locally

Clone the project

```bash
  git clone https://github.com/adan-rodriguez/el-villanense-news.git
```

Go to the project directory

```bash
  cd el-villanense-news
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
